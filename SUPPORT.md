# Getting Support

## Community Support (Free)

- **GitHub Discussions** — ask questions, share ideas: [github.com/myfyio/basefyio/discussions](https://github.com/myfyio/basefyio/discussions)
- **GitHub Issues** — bug reports and feature requests: [github.com/myfyio/basefyio/issues](https://github.com/myfyio/basefyio/issues)
- **Documentation** — guides and API reference: [docs/](docs/)

## Before Asking

1. Check the [README](README.md) Quick Start section
2. Search [existing issues](https://github.com/myfyio/basefyio/issues) and [discussions](https://github.com/myfyio/basefyio/discussions)
3. Run `docker compose logs platform-api` to capture relevant errors

## Response Expectations

This is a community-supported open-source project. Response times vary based on maintainer availability.

For production-critical support with SLA guarantees, see the hosted offering at [basefy.io](https://basefy.io).
