# Changelog

All notable changes to the Anti-Gravity theme by [Hilal Technologic](https://hilaltechnologic.com) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-09-11

### Added

- **Core Theme**
  - Initial release of Anti-Gravity theme
  - Astro 5 + TypeScript + Tailwind CSS setup
  - React integration for interactive components
  - MDX support for blog and documentation

- **Design System**
  - Linear-inspired design tokens (colors, typography, spacing, shadows)
  - Light and dark theme support with system preference detection
  - CSS variables for easy customization
  - Responsive design (320px to ultra-wide)

- **Components**
  - UI Components: Button, Card, Badge, Input, Textarea, ThemeToggle
  - Layout Components: Navbar, Footer
  - Section Components: Hero (4 variants), Features, Pricing, FAQ, CTA, Testimonials
  - Animation Components: Antigravity (3D WebGL particle animation)

- **Pages**
  - Home (SaaS landing with Antigravity animation)
  - Features
  - Pricing
  - About
  - Contact
  - Blog (with MDX support)
  - Documentation (with MDX support)
  - Changelog
  - Roadmap
  - Legal (Privacy Policy, Terms of Service)
  - 404 Error page

- **Features**
  - SEO optimized with meta tags, Open Graph, sitemap
  - Accessibility compliant (WCAG AA)
  - Keyboard navigation support
  - Focus visible states
  - Reduced motion support
  - Content collections for blog/docs

- **Antigravity Animation**
  - 3D WebGL particle animation using Three.js
  - Interactive cursor following
  - 4 particle shapes: capsule, sphere, box, tetrahedron
  - Customizable colors, count, and behavior
  - Graceful fallback for non-WebGL browsers

- **Documentation**
  - README.md with setup instructions
  - THEMING.md for customization guide
  - DECISIONS.md for design decisions
  - QA_CHECKLIST.md for quality assurance
  - Sample blog posts and documentation pages

### Technical Details

- Astro 5.16.8
- React 19.2.3
- Tailwind CSS 3.4.1
- Three.js 0.182.0
- Radix UI primitives
- Lucide React icons
